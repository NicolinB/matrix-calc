# matrix_calc

Matrix calculator written in C supporting matrices up to 8×8.

## Compile & Run

```bash
gcc matrix_calc.c -o matrix_calc -lm
./matrix_calc
```

## Operations

- Addition `A + B`
- Subtraction `A - B`
- Multiplication `A × B`
- Transpose `Aᵀ`
- Determinant (recursive Laplace expansion)
- Scalar multiplication `k × A`
